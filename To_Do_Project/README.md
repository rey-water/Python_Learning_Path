To-Do Лист.
Программа для структурирования дел на день, созданная с целью закрепления знаний в рамках изучения Python.

Особенности:
1) Весь интерфейс реализован на match-case.
2) Для уменьшения количества ошибок в программе добавлена проверка на пустые списки (case 2 и 4), а также используется метод .endswith(), чтобы нельзя было пометить "СДЕЛАНО" несколько раз для одной задачи.

В планах:
Реализовать сохранение задач в файл, чтобы при закрытии программы данные не пропадали (в данный момент такой возможности нет, так как эта тема ещё не изучена).
